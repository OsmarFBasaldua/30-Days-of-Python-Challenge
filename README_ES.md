# 30 Days of Python – Repositorio de Aprendizaje

## Description
Repositorio de aprendizaje personal basado en el reto **30 Days of Python**. Aquí documento mi progreso y ejercicios mientras aprendo los fundamentos de Python paso a paso como principiante.

## Motivation
Decidí comenzar este reto para introducirme al mundo de la programación con Python de forma estructurada y constante. Desde el primer día, el reto me motiva a practicar diariamente y a desarrollar el hábito de escribir código.

## Objective
Aprender los fundamentos de Python, mejorar mi lógica de programación y construir una base sólida a través de la práctica diaria.
